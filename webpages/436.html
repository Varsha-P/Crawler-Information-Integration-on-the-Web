Review created: 12/03/13
customer product rating 1 of 1 found this review helpful.
After only 5 days I'm really starting to like this thing and happy with it. by:
heymikeyd (11 )
This is my first smart phone, so I'm happy to be able to get on the internet with it. It showed up about 5 days ago and I've been messing with it a lot to try to get it all figured out. I bought an unlocked version and I'm using it with Net10 service. I just had to buy a SIM card and a service plan (monthly, not a contract) and configure the Access Point Name (APN). Activation occured within a couple of minutes or less. One of the things that drew me to this phone was the waterproof and shockproof ratings I saw about the phone when I was researching. I'm reading through the 230 page user guide and so far I just see generic information about how I should not get my phone wet or drop it. I expected somewhere to find a statement of the mil spec rating or something. I'll keep looking. However, I'm not planning on testing it by dunking it in the toilet. I had a tough time getting used to the touch screen keyboard, but my younger brother told me I'd get used to it. I am getting better at it, but I prefer the option to talk to the phone and have it converted to text. That feature works surprisingly well. The large 230 page user guide (each page is really only a half page, not a full 8.5"x11" page) is a bit daunting, but I'm learning a lot about how to control the phone by reading it. I recommend downloading it from the Samsung website as it is not included with the phone. The little manual that did come with the phone was pretty much useless. Apparently the jelly bean operating system is available as an upgrade. I have not tried that yet. I wonder how much bigger the file size is compared to the icecream sandwich. Although, everyone knows an icecream sandwich is much bigger than a jelly bean. There is only 8gb hard drive phone storage. I get much better reception with this phone than I did with my previous phone (LG900G). I like the option to use Wi-Fi to save on data downloads through the cell network. The Net10 service allows for 1.5gb of data/month at highspeed. The rest is at low speed. Not sure how easy it is to hit that 1.5gb max. The speaker phone has better sound than my last phone and I like the 32gb external card option. I loaded it with music.
